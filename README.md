- Wan (Leader, Backend Developer)
- Aisyah (Frontend Developer)

## Introduction
SuciTrack is a Laravel-based web application designed to help users track menstrual cycles and purity days in a Shariah-compliant manner.

## Objectives
- Provide an easy-to-use dashboard for cycle records.
- Automate calculation of purity days.
- Support Qada' prayer tracking.

## Features
- User authentication
- CRUD for menstrual records
- Dashboard with cycle status badges
- Reports and analytics
